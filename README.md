# Personal Portfolio

A modern portfolio website built with React and TypeScript. Features custom particle effects and smooth animations with a rhythm game-inspired aesthetic.

## Tech Stack

- React 19 + TypeScript
- Vite
- TailwindCSS v4
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Build for production:
```bash
npm run build
```

## Features

- Custom canvas-based particle system
- Responsive design
- Glass morphism UI elements
- Smooth page transitions
- Mobile-optimized
